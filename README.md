# DIET-AND-FITNESS-RECOMMENDATION-APP
The Diet and Fitness App offers a meal planner, symptom checker, and diet chart sender. Features include emergency response, BMI/health calculators, voice-to-text and text-to-voice, plus a mental health bot for support and a translator for language assistance.
